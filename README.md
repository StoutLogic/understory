# Understory
A WordPress theme layer, built on [Timber](https://github.com/jarednova/timber) and [Twig](http://twig.sensiolabs.org)

> un·der·sto·ry /ˈəndərˌstôrē/ _noun_ - A layer of vegetation beneath the main canopy of a forest.
